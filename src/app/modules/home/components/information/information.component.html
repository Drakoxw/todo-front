Resumen del Proyecto

Aplicación web construida con Angular V20 que permite a los usuarios crear, editar y eliminar Snacks/Elementos y calcular un conjunto de elementos optimos con el minimo de calorias necesarios y el menor peso posible.

Stack Tecnológico
  • • Angular 20 con signals
  • • Abstinencia de ZoneJs
  • • TypeScript
  • • Tailwind CSS
  • • NgPrime
  • • RxJS
Características Clave
  • • Calculo de recursos optimos
  • • Operaciones CRUD
  • • Integración con Back
  • • Api Rest
  • • Validación de datos
