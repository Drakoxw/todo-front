Resumen del Proyecto

Aplicación web construida con Angular V20 que permite gestionar Tareas y generar Reportes de las mismas.

Stack Tecnológico
  • • Angular 20 con signals
  • • Abstinencia de ZoneJs
  • • TypeScript
  • • Tailwind CSS
  • • NgPrime
  • • RxJS
Características Clave
  • • Resumen/Estado de las tareas
  • • Operaciones CRUD
  • • Integración con Back
  • • Api Rest
  • • Validación de datos
